node_extended.node_id (n)
Get the ID of a node. We have to convert the node into a string and than have to extract the ID from this string using a regular expression. If you convert a node into a string it looks like: | nil : hlist 2>|.

Parameters:

  • n
tree.format_attributes (head)
Attributes are key/value number pairs. They are printed as an inline list. The attribute |0| with the value |0| is skipped because this attribute is in every node by default.

Parameters:

  • head
tree.set_state (level, connection_type, connection_state)
|level| is a integer beginning with 1. The variable |connection_type| is a string, which can be either |list| or |field|. The variable |connection_state| is a string, which can be either |continue| or |stop|.

Parameters:

  • level
  • connection_type
  • connection_state
